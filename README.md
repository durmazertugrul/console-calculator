Console Calculator (C#)

A simple and user-friendly console-based calculator built with C#.
This project demonstrates basic programming concepts such as loops, conditionals, and object-oriented design.

---

🚀 Features

➕ Addition 
➖ Subtraction 
✖️ Multiplication 
➗ Division (with zero-check) 
🔢 Exponentiation 
√ Square Root 
％ Modulo 
📜 Operation History 
🧹 Clear History 
🔁 Continuous usage with menu system 

---

🧠 Concepts Used

* `while` loop for continuous execution
* `switch-case` for operation selection
* `if-else` for input control
* Custom `HistoryItem` class
* `List<T>` for storing operation history

---

🛠️ How It Works

1. User selects an operation from the menu
2. Inputs required number(s)
3. Program calculates the result
4. Result is displayed
5. Operation is stored in history
6. User can view or clear history anytime

---

▶️ How to Run

1. Open the project in Visual Studio
2. Build the solution
3. Run the program (Ctrl + F5)

---

📌 Example

```
Select an operation: 1
Enter first number: 5
Enter second number: 3
Result: 8
```
